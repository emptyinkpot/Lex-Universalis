## 📥 下载与使用指南

### 下载步骤（以 Flaticon 为例）

1. **访问网站**：https://www.flaticon.com/
2. **搜索图标**：输入关键词（如 `medieval shield`）
3. **筛选免费图标**：
   - 点击 "Filter"
   - 选择 "Free"
   - 勾选 "Commercial use allowed"
4. **下载图标**：
   - 选择尺寸（推荐 256px 或 512px）
   - 点击 "Download"
   - 选择 PNG 或 SVG 格式

### 放入项目目录

```bash
# 1. 下载图标到本地
# 例如：下载 lion_crest.png

# 2. 创建图标目录
mkdir -p client/assets/icons/factions
mkdir -p client/assets/icons/ui
mkdir -p client/assets/icons/cards

# 3. 复制图标到对应目录
# 阵营图标
cp ~/Downloads/lion_crest.png client/assets/icons/factions/england.png
cp ~/Downloads/fleur_de_lis.png client/assets/icons/factions/france.png
cp ~/Downloads/golden_eagle.png client/assets/icons/factions/hre.png
cp ~/Downloads/dragon_ship.png client/assets/icons/factions/viking.png
cp ~/Downloads/purple_cross.png client/assets/icons/factions/byzantine.png

# UI 图标
cp ~/Downloads/swords.png client/assets/icons/ui/attack.png
cp ~/Downloads/shield.png client/assets/icons/ui/defense.png
cp ~/Downloads/coin.png client/assets/icons/ui/gold.png
cp ~/Downloads/fist.png client/assets/icons/ui/influence.png

# 卡牌类型图标
cp ~/Downloads/knight.png client/assets/icons/cards/unit.png
cp ~/Downloads/magic_book.png client/assets/icons/cards/spell.png
cp ~/Downloads/castle.png client/assets/icons/cards/building.png
```

### 在代码中使用

```typescript
import { Image } from 'react-native';

// 方式1：使用 require
<Image 
  source={require('@/assets/icons/factions/england.png')} 
  style={{ width: 48, height: 48 }} 
/>

// 方式2：封装阵营图标组件
import englandIcon from '@/assets/icons/factions/england.png';
import franceIcon from '@/assets/icons/factions/france.png';

const FactionIcon = ({ faction, size = 48 }) => {
  const icons = {
    ENGLAND: englandIcon,
    FRANCE: franceIcon,
    HRE: hreIcon,
    VIKING: vikingIcon,
    BYZANTIUM: byzantineIcon,
  };

  return (
    <Image 
      source={icons[faction as keyof typeof icons]} 
      style={{ width: size, height: size }} 
    />
  );
};
```
