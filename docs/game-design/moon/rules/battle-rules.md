# 战斗规则

这一份是从《临时约法 2.0》中抽出来的战斗页速查版，供 UI、战斗系统和后续数值调整共同引用。

## 核心阶段

- 声明阶段：先声明行动和目标，允许反制、打断和前置判定。
- 施行阶段：效果实际结算，伤害、治疗、弃牌和状态变化在这里落地。
- 结束阶段：收尾、移除临时效果、推进回合和触发回合结束事件。

## 槽位与攻击

- 每名玩家默认操控 3 名前排角色与 3 名后排角色。
- 前排角色可以进行基础攻击。
- 后排角色不能进行基础攻击，也不能被基础攻击直接命中。
- 基础攻击的骰点结算：
  - 1、2、3：造成 1 点伤害
  - 4、5：造成 2 点伤害
  - 6：本次攻击 miss

## 反制与打断

- 携带“反制”字样的卡可以在目标锁定后插入结算。
- 反制卡如果与来袭效果同类，则优先结算并令来袭效果无效。
- 反制本身结算完后通常进入弃置。

## 月相规则

- 月相卡在游戏开始与每轮开始时结算。
- 月相影响全体角色，且不可被反制。
- 月相分为白月、蓝月、红月、黑月，对摸牌、蓝量、伤害和移动都有全局影响。

## UI 提示建议

- 战斗页应始终显示当前阶段、前后排状态和是否可反制。
- 受击、反制、回合推进都应该有明显的视觉和触觉反馈。
- 战斗日志要短，优先保留最近一到三条结算信息。
