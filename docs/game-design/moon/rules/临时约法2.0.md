# 临时约法2.0

牌篇

角色牌：每名角色的血与蓝总计15，血量归零后即退场，可以通过消耗蓝量点数来抽取法术卡，同时部分角色卡可以通过消耗蓝量点来使用角色自带技能。每名角色都有特殊的技能（注：人物卡技能不建议设置得过于复杂）。每名角色的法术系别固定，一般有一到两种属性，法术卡只能从该角色携带的属性里抽取。

法术卡：消耗蓝量点以抽取，如无特殊说明，其他角色技能、普通卡效果、法术卡效果不能作用于玩家手中的法术卡。法术卡分为四种类型，风，林，火，山（风，主机制，效果和反制，林，主要作用于其他卡，火，主伤害，山，主防御。提供一个大概思路，实际设计可以不遵循这种思路，比如山系法术也可以有主伤害的效果。）

普通卡：每回合抽取，普通卡中包括 行动卡，道具卡，诡计卡，事件卡，通用卡

行动卡，行动

道具卡，置于道具区，每回合道具区只能放置一张道具卡，然后根据道具卡的效果在回合结束阶段根据卡的效果决定卡片的去向。

诡计卡，诡计

事件卡，抽出即触发，一般对全体玩家使用，无发起人

通用卡，可以理解为简易版的法术卡

特殊卡片机制

反制，携带反制字样的卡片可以盖置于反制区，直到下一次自己回合开始阶段弃于牌堆。反制卡可以在其他玩家选定自己为目标后使用，此时如果你的反制卡的卡牌类型与其相同，则该玩家对你发动的卡片效果无效，并且该反制卡的效果在回合外视为由你发动，并且该卡的结算始终先于引起反制效果的卡片的结算。该过程结束后，反制卡被弃置。

月——在游戏开始时与每一轮回合开始时抽取，效果无条件作用于所有角色，无法被反制，不视为由任何玩家发动，月的效果作用于该轮

白月，发动时，所有人摸一张普通牌，然后所有玩家可以移动一个角色；替换时，所有人摸一张普通牌

蓝月，发动时，所有人恢复一点蓝量；在发动阶段，法术卡不计入手牌上限

红月，所有人，失去一点血量；在发动阶段，当任意玩家杀死一名角色后可以在本回合结束后获得一个额外回合；当你在该阶段第一次对其他角色造成伤害时，你回复一点血量

黑月，发动时，所有人弃置一张普通牌，然后所有玩家可以移动一个角色；替换时，所有人弃置一张普通牌

大概任意效果都可以分为3个阶段

声明阶段，一部分卡片或者效果可以再声明阶段触发，打断其效果或者发动自己的效果

施行阶段，即效果的实现以及各方面结算阶段（结算阶段也应该可以分为三个阶段，但一般不建议在这个阶段作太多文章，但是伤害抵免可以算在这里，主要可能会有伤害判定前和判定后的阶段，而判定阶段其实也可以分为三个阶段）

结束阶段。

如上，三个阶段主要是为了明晰一些效果的触发，分为前后是为了方便其它角色或卡牌的效果在使用时没有异议，某张卡的三个阶段中最重要显而易见为第二阶段，其它阶段主要留给其它卡施展效果。

目标，杀死除自己外所有玩家

每名玩家操控三名角色，一般而言，己方的行动需要有一个明确的声明单位，方便其它反馈的进行。而在法术卡的释放判定过程中，发起人只能是与法术卡的属性一致的角色，但是可以并非由该角色的蓝量抽出的法术卡。

另外，每名前排角色都可以在回合内对其它玩家的任意一名前排角色进行一次基础攻击，根据投骰子数判定伤害：

1）1，2，3——一点伤害

2）4，5——两点伤害

3）该次攻击视为被miss

后排角色无法基础攻击，同时无法被基础攻击

玩家可以在游戏开始阶段决定自己角色的站位，之后的游戏中必须依靠技能或卡片效果进行改动

手牌上限：暂定手牌上限为——普通牌为6-x（x为重伤或死亡角色数）

——法术卡为3-x（x为死亡角色数）

摸牌：——普通牌，每回合定为3张

——法术卡，每回合每名角色最多主动摸取两张法术卡，但是每名玩家每回合最多主动抽取四张

短：本回合结束阶段最后弃置

中：每回合开始阶段顺时针旋转一次，旋转一周即旋转至原来位置时立即弃置
